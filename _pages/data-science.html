Data Science Practitioner Training BDK Testing

Data scientists are needed to find insights and to solve meaningful business problems with GSA. Get the real-world knowledge and hands-on experience that can help you excel in data science.

Earning the GSA Data Science Certificate has several benefits:

  • Showcasing Credibility
  • Enhancing Skillsets
  • Opening Doors to New Opportunities

Those who meet GSA's high standards and finish the final capstone distinguish themselves and create greater opportunities for career enhancement.

Individuals could consider themselves an expert in just one area or multiple areas. Students will choose from 2 available programming languages, either Python or R. This decision should be based on the needs and requirements of your business domain. For assistance in help with making this choice, please contact us.

Interested?

If you are interested in participating in one of our two pilot cohorts, please get your supervisors permission and fill out the form below. You will be contacted individually if you are chosen to participate. Course instruction will be instructor-led and hands-on with delivery conducted both in-person, as well as virtual.

Certificate Courses

  • Data Science Overview

    The growing field of data science is emerging at the intersection of social science and statistics, information and computer science, and design. This course takes a broad look at data science focusing on providing you with an overview of data science, terms, processes, roles and workflows. You'll gain some insight as to why some data scientists use R and others prefer Python. Students will be introduced to a sample data science project and consider how they might create their own projects to add value to their teams within GSA.

    View class detail

    Python for Data Science - OLU

    This course introduces the concepts of data science and provides a brief overview of the Python skills needed to work with data. In this course, you will learn about IPython components, Notebook, and the NumPy module. There's still more to come as the course guides you toward managing financial statistics with financial big data.

    View class detail

    Fundamental Methods for Data Science in R - OLU

    R is a free software environment for statistical computing and graphics and has become an important tool in modern data science. In this course, you will learn the fundamental R methods that data scientists use in their everyday work.

    View class detail

  • Intro to Programming in Python

    Python is a general-purpose programming language that is extremely popular for data science. In this introduction to programming with Python, you'll learn and practice key computer science fundamentals (such as expressions, variables, and data types) and then apply these elements to create your first Python program. The class is designed for students with very little or no computing background.

    View class detail

    Intro to Programming in R

    R is a language and environment for statistical computing and graphics. In this introduction to programming with R, you'll learn and practice key computer science fundamentals (such as expressions, variables, and data types) and then apply these elements to create your first R program. The class is designed for students with very little or no computing background.
